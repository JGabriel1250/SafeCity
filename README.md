# SafeCity 🗺️

Mapa interativo de segurança urbana por estado, cidade e bairro.

## Tecnologias usadas
- HTML, CSS, JavaScript
- Leaflet.js (mapa interativo)
- API do IBGE (estados e cidades)

## Como usar
1. Selecione o estado
2. Selecione a cidade
3. Selecione o bairro
4. Clique em Buscar

## Objetivo
Projeto educacional para aprender consumo de APIs e mapas interativos.
