# 🛡️ SafeCity

Aplicação web interativa para visualização de locais e simulação de dados de segurança urbana em mapa.

🔗 **Acesse o projeto:**  
https://jgabriel1250.github.io/SafeCity/

---

## 🚀 Sobre o projeto

O **SafeCity** é um projeto desenvolvido com foco em aprendizado prático de desenvolvimento web, integrando:

- Consumo de APIs externas
- Manipulação do DOM
- Eventos em JavaScript
- Visualização geográfica com mapas

O sistema permite selecionar **estado, cidade e bairro**, localizar no mapa e exibir informações simuladas de segurança.

---

## 🧠 Funcionalidades

- 📍 Busca de localização por estado, cidade e bairro  
- 🗺️ Exibição de mapa interativo com zoom automático  
- 📌 Marcação dinâmica do local selecionado  
- 🔴 Destaque da área com GeoJSON (quando disponível)  
- ⚪ Fallback com círculo quando não há polígono  
- 📊 Exibição de dados simulados de ocorrências  
- 🔄 Atualização dinâmica sem recarregar a página
